# Cloud-Project2
This cloud project involves hosting a sample website over s3 and cloud front, usually companies deploy their frontend website like this 

CloudFront provides the following :
✅ HTTPS automatically (free SSL certificate via ACM)
✅ Fast global delivery via CDN
✅ S3 bucket kept private — only CloudFront can read it
✅ Custom error pages
✅ Cache invalidation when you update your site
